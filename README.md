# Week2-Object-Oriented-Programming

This repository contains Java programming exercises focusing on Object-Oriented Programming (OOP) concepts.

## 📁 Folder Structure

* `01 Java Class and Object`: Introduction to classes and objects in Java.
* `02 Java Constructor`: Understanding constructors and their usage.
* `03 this, static, final`: Exploring the `this` keyword, static members, and final modifiers.
* `04 Object Modelling`: Techniques for modeling real-world entities using Java classes.
* `05 Inheritance`: Implementing inheritance to promote code reusability.
* `06 Polymorphism`: Demonstrating method overloading and overriding for polymorphic behavior.

## 🧑‍💻 Getting Started

To run the Java files:

1. Ensure Java is installed on your system.
2. Navigate to the desired folder.
3. Compile the Java file:

   javac FileName.java

4. Run the compiled class:

java FileName



## ✅ Prerequisites

* Java Development Kit (JDK) 8 or higher
* A code editor like VS Code, IntelliJ IDEA, or Eclipse
